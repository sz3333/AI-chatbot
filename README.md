# Gemini AI Bot

Бот на Aiogram 3.x + Google Gemini API с памятью, личными промтами и админ-панелью.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

Вариант 1 — через переменную окружения:
```bash
BOT_TOKEN=ваш_токен python gemini_bot.py
```

Вариант 2 — создай файл `.env` рядом с ботом:
```
BOT_TOKEN=ваш_токен_бота
```
Затем просто:
```bash
python gemini_bot.py
```

## Команды

### Для всех (только в ЛС)
| Команда | Описание |
|---|---|
| `/start` | Начать работу |
| `/help` | Справка |
| `/setprompt <текст>` | Установить личный промт (влияет на стиль ответов лично для тебя) |
| `/clearprompt` | Сбросить личный промт |
| `/clearhistory` | Очистить историю диалога |
| `/showhistory` | Показать последние 20 сообщений истории |

### Для владельца (ID: 93823977)
| Команда | Описание |
|---|---|
| `/settokens key1,key2,key3` | Установить Gemini API ключи (через запятую) |
| `/setglobalprompt <текст>` | Глобальный промт для всех пользователей |
| `/statdb` | Статистика БД (юзеры, история, ключи, модель) |
| `/addusers` | Добавить юзеров из .txt файла (только в ЛС) |
| `/broadcast <текст>` | Рассылка всем активным пользователям |
| `/clearall` | Очистить всю историю всех юзеров |
| `/setmodel <модель>` | Сменить модель Gemini |
| `/keystat` | Статус API ключей (ошибки по каждому) |

## Формат файла для /addusers

Файл `users.txt` — каждый Telegram ID на новой строке:
```
123456789
987654321
555000111
```
или через запятую:
```
123456789, 987654321, 555000111
```

## Работа в группах

В группах бот отвечает только если:
- Его упомянули через `@username_бота`
- Ответили (reply) на его сообщение

В личных сообщениях — отвечает на всё.

## Рекомендуемые модели Gemini

- `gemini-2.5-flash` — быстрый, бесплатный (по умолчанию)
- `gemini-2.5-pro` — умнее, медленнее
- `gemini-2.0-flash` — стабильный вариант
